# Introduction

MimDB is a self-hosted Backend-as-a-Service platform. It provides authentication,
database management, real-time subscriptions, file storage, and a REST API -
all from a single binary.

## Features

- **Auth** - Email/password signup, JWT tokens, session management
- **Database** - Multi-tenant PostgreSQL with schema isolation
- **REST API** - Auto-generated CRUD endpoints via PostgREST
- **Realtime** - WebSocket subscriptions for database changes
- **Storage** - File uploads with resumable (tus) protocol and signed URLs
- **Extensions** - pgvector, pg_cron, pg_stat_statements, and more

<Note>
MimDB is currently in active development. APIs may change between releases.
</Note>
